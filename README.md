# `~/.zshrc`

This is my `~/.zshrc` for various systems.

## Installation

### macOS

Clone repo to `~/src/zshrc`:

```zsh
cd
mkdir -p ~/src/
cd ~/src/
git clone git@github.com:ctsrc/zshrc.git
```

Symlink.

```zsh
ln -s ~/.zshrc src/zshrc/macOS.zshrc
```
